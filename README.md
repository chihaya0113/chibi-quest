# ちびクエ

小学3年生向けの家庭内学習アプリです。

Phase1 は算数・国語・社会・英語・理科です。

- 実行時AIなし
- 固定問題DB
- 1回10問
- 経験値・レベル・結果画面
- 親が単元を解放できる構造

## 起動

```bash
npm run serve
```

ブラウザで `http://localhost:5173` を開きます。

## 問題生成

```bash
npm run generate:questions
```

生成される問題数は算数260問、国語260問、社会260問、英語260問、理科260問です。

## 問題レビュー

```bash
npm run audit:questions
```

問題数、必須フィールド、4択の重複、正解の有無、説明文、問題文パターンの重複を確認します。
